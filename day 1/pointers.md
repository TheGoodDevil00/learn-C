# Pointers
#### If we think of memory as a very big array of bytes(8 bits).
#### Data is stored in such array, if the number of bytes required is greater than 1, as is the case most often, it is stored in multiple bytes.
#### Since we defined memory as an array, each of the constituent byte can be referred to by it's index.
#### This index is also called an address or a location.
#### The variable in which this address is stored is known as a POINTER.fi
